GIT Y GITHUB



Repositorio Web GitHub




1. Ingresar al portal GITHUB Aquí


2. Crear usuario de acceso al portal


3. Iniciar sesión con el usuario creado.


4. Dirigirse a la parte superior derecha donde están los repositorios y cliquear en el botón Nuevo.


5. Dar un nombre al repositorio luego cliqueamos en el botón crear repositorio.

Repositorio Local GitHub




1. Descargar gitHub Desktop desde Aquí


2. Realizar instalación de GitHub Desktop.


3. Crear usuario de acceso al portal. Desde el GitHub Desktop Re-direccionara al portal web para la creación del usuario.


4. Iniciar sesión con el usuario creado.


5. Despupés de iniciar sesión nos dirigimos a la opción crear nuevo repositorio.
